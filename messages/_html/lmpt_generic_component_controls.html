MAVLink Commands (MAV_CMD)

MAVLink commands (MAV_CMD) and messages are different! These commands define the values of up to 7 parameters that are packaged INSIDE specific messages used in the Mission Protocol and Command Protocol. Use commands for actions in missions or if you need acknowledgment and/or retry logic from a request. Otherwise use messages.

MAV_CMD_START_NEW_SAVED_MEDIA_SESSION (56025 )

[Command] Starts a new session or directory for logs or saved media.

Param (:Label) Description

MAV_CMD_SET_COMPONENT_IP_SETTINGS (56026 )

[Command] Change component IP settings or set DHCP mode. The payload should respond with a COMPONENT_IP_SETTINGS message; use it to confirm command success.

Param (:Label) Description Values
1: Network Setting Params 2 and 3 are ignored if this enables DHCP. NETWORK_SETTINGS
2: Static IPv4 Address Static IPv4 address. Use a network-style (Big-Endian) uint32_t, never interpret the value as a float. Only used if DHCP is disabled.
3: Static Subnet Mask Static IPv4 subnet mask. Use a network-style uint32_t (Big-Endian), never interpret the value as a float. Only used if DHCP is disabled.

MAV_CMD_SET_UPDATE_MODE (56027 )

[Command] Tell a component to enter/exit an Update Mode. What that specifically means would of course be specific to the application or component. The idea is to provide some message to tell a component to change its behavior to facilitate some sort of software update.

Param (:Label) Description
1: Update Mode 1: Enter update mode. 0: Exit update mode, i.e. resume normal operation.

MAVLink Messages

COMPONENT_IP_SETTINGS ( #156025 )

[Message] (MAVLink 2) Reports the component's IP settings, e.g. current IPv4 addr/netmask, and whether use of DHCP is enabled.

Field Name Type Values Description
current_ipv4_address uint32_t The component's IPv4 address. Use a network-style (Big-Endian) uint32_t, never interpret the value as a float.
current_subnet_mask uint32_t The component's IPv4 subnet mask. Use a network-style uint32_t (Big-Endian), never interpret the value as a float.
network_setting uint16_t NETWORK_SETTINGS The component's network settings, e.g. whether DHCP is enabled.

COMPONENT_VERSION_INFORMATION ( #156026 )

[Message] (MAVLink 2) Version information for a software component.

Field Name Type Description
name char[32] Name of the software component
version char[32] Version of the software component
hash char[8] Version commit hash digest.

PARTITION_STATUS_INFO ( #156027 )

[Message] (MAVLink 2) Reports the current state of a storage partition.

Field Name Type Values Description
storage_id uint8_t Matches the storage_id field from this partition's STORAGE_INFORMATION message.
state uint8_t PARTITION_STATE Status of this partition.
command_reply uint8_t PARTITION_COMMAND The PARTITION_COMMAND this was sent in response to, or 0 if this message was sent automatically.
command_result uint8_t PARTITION_COMMAND_RETURN_VALUE The result of the PARTITION_COMMAND, in case it can't be inferred from the partition's state.

PARTITION_COMMAND_PACKET ( #156028 )

[Message] (MAVLink 2) Gives a PARTITION_COMMAND to a storage partition. Implemented as a message rather than a MAV_CMD to include encryption passphrases, which are too big to fit in a COMMAND_LONG. The component should respond with a PARTITION_STATUS_INFO message.

Field Name Type Values Description
target_system uint8_t System ID
target_component uint8_t Component ID
storage_id uint8_t The specific partition ID, in case the component has multiple.
partition_cmd uint8_t PARTITION_COMMAND The PARTITION_COMMAND to execute.
new_passphrase char[64] New passphrase for an encrypted partition. Terminate with a NULL byte if less than 64 characters. Required for PARTITION_CMD_MAKE_SECURE_PARTITION, PARTITION_CMD_ADD_KEY, and PARTITION_CMD_CHANGE_KEY.
existing_passphrase char[64] Passphrase already in use by an encrypted partition. Terminate with a NULL byte if less than 64 characters. Required for PARTITION_CMD_MOUNT (if the partition is encrypted), PARTITION_CMD_ADD_KEY, PARTITION_CMD_DELETE_KEY, and PARTITION_CMD_CHANGE_KEY.

PROTECTED_PARAM_EXT_REQUEST_READ ( #156029 )

[Message] (MAVLink 2) Request to read the value of a password-protected parameter (see PARAM_EXT_REQUEST_READ). Responds with a PROTECTED_PARAM_EXT_VALUE if correct password is provided, otherwise responds with PROTECTED_PARAM_EXT_ACK, with empty parameter_id and PARAM_ACK_FAILED.

Field Name Type Description
target_system uint8_t System ID
target_component uint8_t Component ID
param_id char[16] Parameter ID, terminated by NULL if the length is less than 16 characters.
param_index int16_t Parameter index. Set to -1 to use the Parameter ID field as identifier (else param_id will be ignored)
param_password char[64] Password for accessing protected parameters.

PROTECTED_PARAM_EXT_REQUEST_LIST ( #156030 )

[Message] (MAVLink 2) Request all password-protected parameters of this component (see PARAM_EXT_REQUEST_LIST). If correct password is provided, the component will respond with a PROTECTED_PARAM_EXT_VALUE for each parameter. Otherwise responds with PROTECTED_PARAM_EXT_ACK, with empty parameter_id and PARAM_ACK_FAILED.

Field Name Type Description
target_system uint8_t System ID
target_component uint8_t Component ID
param_password char[64] Password for accessing protected parameters.

PROTECTED_PARAM_EXT_VALUE ( #156031 )

[Message] (MAVLink 2) Emit the value of a protected parameter (see PARAM_EXT_VALUE). This is only emitted in response to a PROTECTED_PARAM_EXT_REQUEST_READ or PROTECTED_PARAM_EXT_REQUEST_LIST that provided a valid password.

Field Name Type Values Description
param_id char[16] Parameter id, terminated by NULL if the length is less than 16 characters.
param_value char[128] Parameter value
param_type uint8_t MAV_PARAM_EXT_TYPE Parameter type.
param_index uint16_t Index of this parameter
param_report_count uint16_t Total number of parameters being reported (e.g. 1 if in response to PROTECTED_PARAM_EXT_REQUEST_READ, probably more than 1 if in response to PROTECTED_PARAM_EXT_REQUEST_LIST). This may differ from the total number of protected parameters, as different parameters may require different passwords.

PROTECTED_PARAM_EXT_SET ( #156032 )

[Message] (MAVLink 2) Set a protected parameter's value (see PARAM_EXT_SET). Can be used within a PARAM_TRANSACTION. Setting the param value will fail if the wrong password is provided.

Field Name Type Values Description
target_system uint8_t System ID
target_component uint8_t Component ID
param_id char[16] Parameter id, terminated by NULL if the length is less than 16 human-readable characters.
param_value char[128] Parameter value
param_type uint8_t MAV_PARAM_EXT_TYPE Parameter type.
param_password char[64] Password for accessing protected parameters.

PROTECTED_PARAM_EXT_ACK ( #156033 )

[Message] (MAVLink 2) Response from a PROTECTED_PARAM_EXT_SET message, or a failed PROTECTED_PARAM_EXT_REQUEST_READ or PROTECTED_PARAM_EXT_REQUEST_LIST.

Field Name Type Values Description
param_id char[16] Parameter id, or empty string if in response to a failed PROTECTED_PARAM_EXT_REQUEST_READ or PROTECTED_PARAM_EXT_REQUEST_LIST.
param_value char[128] Parameter value (new value if PARAM_ACK_ACCEPTED, current value otherwise)
param_type uint8_t MAV_PARAM_EXT_TYPE Parameter type.
param_result uint8_t PARAM_ACK Result code.
